#  Realtime battleships game with chat #

## Technology stack ##

> Django + channels
> 
> JavaScript
> 
> HTML + CSS
> 
> Redis
> 
> Docker

## Set up ##
1. Clone project 

`git clone git@gitlab.com:tovarischduraley/seabattle.git`

2. Change directory

`cd SeaBattle`

3. Change CRLF eol to LF in `entrypoint.sh`
4. Build docker-compose
`docker-compose build`
5. Run docker-compose
`docker-compose up`
